# 🩺 Healthcare Chatbot MDP Decision Dashboard

An interactive dashboard to visualize and solve Markov Decision Processes (MDP) for strategic healthcare interventions.

### Features
- **Real-time Policy Iteration**: Recalculates the optimal treatment path as you adjust variables.
- **Dynamic Visualizations**: Uses Graphviz to render state-transition diagrams.
- **Executive Reporting**: Generates a downloadable PDF summary of the calculated strategy.

### How to Run Locally
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Launch: `streamlit run app.py`
